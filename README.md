# surf-cam
Embedded computer vision surf tracking camera, YOLOv3 with darknet on NVIDIA Jetson Nano and streaming with FFmpeg (Wi-Fi and LTE)

## Install `darknet`

```
git clone https://github.com/pjreddie/darknet.git
cd darknet
make
```
